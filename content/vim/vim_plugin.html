Vim 腳本函式庫

vim-scripts/L9: Vim-script library - GitHub

查找文件

vim-scripts/FuzzyFinder: buffer/file/command/tag/etc explorer with fuzzy matching - GitHub

依賴： Vim 腳本函式庫 。

Plug 'vim-scripts/FuzzyFinder'

    nmap Ff :FufFile
    nmap Fb :FufBuffer

程式物件整理

majutsushi/tagbar: Vim plugin that displays tags in a window, ordered by scope - GitHub

需額外安裝 ctags 。

Plug 'majutsushi/tagbar', { 'on': 'TagbarToggle' }

    nmap <F8> :TagbarToggle<CR>

標記減量預覽

iamcco/markdown-preview.vim: Real-time markdown preview plugin for vim - GitHub

由於筆者是以虛擬機上使用 Vim， 所以才會使用修改原作後的插件。

Plug 'BwayCer/markdown-preview.vim', { 'for': 'markdown' }

    nmap <F9> :MarkdownPreview<CR>
    nmap mdstop :MarkdownPreviewStop<CR>

文本除錯引擎

w0rp/ale: Asynchronous Lint Engine - GitHub

需等到 Vim 8 升級後才能安裝。

" Plug 'w0rp/ale'
" set statusline+=%5*『\ %{exists('g:loaded_ale')?ALEGetStatusLine():''}』%*
" set statusline+=%5*%{exists('g:loaded_fugitive')?fugitive#statusline():''}%*
